# Tampa to Los Angeles road trip

Interactive, mobile-friendly map for a July 9–17, 2026 road trip through Kentucky, Texas, Route 66, Monument Valley and Nevada's Extraterrestrial Highway.

Open `index.html` locally or visit the GitHub Pages site. The map uses Leaflet, OpenStreetMap tiles and OSRM road routing, so an internet connection is required for the basemap and calculated route.

Destination photographs link to their Wikimedia Commons source and licensing information. Always verify reservations, weather, closures and operating hours before departure.
